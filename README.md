# Shifts App - جدول الـورادي v1.0.0

تطبيق احترافي لإدارة الورادي والإجازات والأذونات لموظفي (حديد عز)، مصمم ليعمل بكفاءة عالية وبدون إنترنت.

## المميزات الرئيسية (v1.0.0) 🚀
- **نظامي عمل (نهاري / ورادي):** دعم كامل للدوام الدوري (أ، ب، ج، د) والدوام الثابت (النهاري).
- **أيام الراحة المخصصة:** إمكانية تحديد أيام الراحة الأسبوعية في نظام النهاري وحفظها بشكل دائم.
- **إدارة الإجازات والأذونات:** واجهة حديثة لتسجيل الإجازات السنوية، العارضة، المرضية، وأذونات الانصراف بالوقت.
- **الرموز اللونية:** تمييز الأذونات باللون البرتقالي الجذاب لتسهيل مراجعة الأرصدة.
- **العمل أوفلاين (100% Offline):** التطبيق لا يحتاج لإنترنت؛ جميع الخطوط والبيانات مخزنة محلياً.
- **تاريخ هجري وميلادي:** تحديث ذكي للتواريخ مع إمكانية الضبط اليدوي للتاريخ الهجري.
- **الخصوصية والأمان:** يتم حفظ جميع البيانات على جهازك فقط عبر `LocalStorage`.
- **النسخ الاحتياطي:** إمكانية تصدير واستيراد البيانات يدوياً وتلقائياً على ذاكرة الهاتف.

## التقنيات المستخدمة 🛠️
- **Frontend:** React + TypeScript + Tailwind CSS.
- **Mobile Integration:** Capacitor (لتحويل المشروع إلى تطبيق Android/iOS).
- **Icons:** Lucide React.
- **Fonts:** Cairo (Localized).
